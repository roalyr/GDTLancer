# GDTLancer

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

**GDTLancer** is a space simulation game that combines 3D space flight with automated tabletop roleplaying game (TTRPG) mechanics.

## The Setting

The game is set in an isolated, early-stage frontier. There are no large governments, corporations, or galactic navies. Human society is divided into small, independent **Communities** living on isolated stations and moons. Spacecraft are massive, resource-heavy industrial machines, and pilots are rare. The game focuses on community survival, local disputes, and the difficult logistics of space travel, rather than heroic space opera.

## Two-Speed Gameplay

The game alternates between two modes:
- **Flight Mode:** Real-time 3D space flight where players navigate between locations.
- **Chronicle Mode:** A paused, menu-driven TTRPG interface where the game engine handles the math and rules, and the player authors the narrative.

## Core Design Rules

1. **The Automated Engine:** In Chronicle Mode, the game engine only provides mechanical data (numbers, tags, and menus). It never writes story text, dialogue, or scene descriptions.
2. **Player Authorship:** The player is the only one who writes the story. Using the **Narrative Template Logbook**, the player selects system-generated keywords and optionally adds their own text to record what happened. This log is saved permanently to the game's Chronicle.

### Technology Stack
* Primary Engine: **Godot Engine 3**

## CURRENT GOAL: NPC Dock-Market Exchange And Dynamic Market Quantities
- TARGET_SCOPE: Make NPC agents interact with station `market_inventory` through the same bounded buy/sell seam the player already uses, so market quantities and agent credits change dynamically from NPC docking trade, not only from player actions. NPC dock-trade must route through existing `available_services` gating (lawful `trade` vs. illicit `black_market`) using the live legality context already derived by `BridgeSystems` + `AffinityMatrix`. NPC dock-trade is a qualitative cargo-to-market exchange at the docked station using the same `market_inventory` shape, bounded by the NPC's existing cargo tag and the station's available quantities.
